iScript
iScript is a visual scripting framework for Minecraft Forge 1.20.1. Create dialogs, event graphs, quests, cutscenes, NPCs and regions — all through in-game editors with node-based workflow.
Features
Dialog Editor — node-based conversation builder with branching, conditions and actions.
Event Graph — visual scripting for game logic: triggers, delays, loops, random branches.
Quest Editor — full quest pipeline with stages, objectives and rewards.
NPC Editor — configure appearance, AI, factions, combat stats and trading.
Cutscene Editor — timeline-based camera paths with splines and effects.
Region Editor — define areas with enter/exit/tick effects.
Status
This is an early alpha (0.0.1-alpha). Expect bugs and missing polish. Feedback is welcome!
Installation
Download the latest jar from CurseForge / GitHub Releases.
Drop it into your mods folder alongside Forge 1.20.1.
Launch the game and press the dashboard key (configurable) or use the Script Block.
